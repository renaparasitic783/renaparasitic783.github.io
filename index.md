## ⚠️ Troubleshooting Typical Errors

If the installer freezes or throws an error code, use these solutions.

### Installation Failed: Access Denied
The installer lacks permission to create folders. Return to your downloads folder and run the file as an administrator. Ensure you are signed into your Windows profile as an administrator user.

### Setup Stops at 99 Percent
This typically happens when Windows Defender interrupts the process. Check your Windows Notifications for a block alert. Choose "Allow on device" or "Run anyway" when prompted.

### Missing DLL Files
If the software complains about missing dynamic link libraries, your computer needs the latest Visual C++ Redistributables. Download these from the official Microsoft support page.

### The Application Does Not Open
This happens if the installation path contains special characters or if the software did not register properly. Uninstall the software, restart your computer, and run the install process a second time in a different folder, such as C:\ArtRage.
